# Angular Product Dashboard 🚀

An enterprise-grade product dashboard built using Angular 21, demonstrating modern frontend architecture, state management using signals, and performance-optimized UI patterns.

---

## 🔥 Features

- Product listing using real API (DummyJSON)
- Search with real-time filtering
- Sorting (price, rating)
- Client-side pagination
- Responsive grid layout
- Modern Angular control flow (@for, @if)
- Signals-based state management
- Clean and scalable architecture

---

## 🧠 Tech Stack

- Angular 21 (Standalone APIs)
- Signals (state management)
- RxJS
- Tailwind CSS
- REST API (DummyJSON)

---


---

## ⚡ Key Concepts Demonstrated

- Signals for reactive state
- Computed properties for filtering & pagination
- Separation of concerns (API → Store → UI)
- Performance optimization using immutable operations
- Scalable folder structure
